# Einführung

Ziel der Bachelorarbeit ist es, ein client-server-basiertes System zur Verbesserung der Genauigkeit des Standard-GPS zu entwickeln.  
Das System besteht aus drei funktionellen Einheiten:  

- **Mobile Applikation**  
- **Server**  
- **Referenzstation**  

In diesem Repository befindet sich der **Mobile Applikation**.


Nutzung vom Service ist nur unter Android möglich, da IOS stellt die API der Öffentlichkeit nicht zu verfügung.
